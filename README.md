# Flight Timezone App

フライト中の現在地・出発地・到着地のローカル時刻を地球儀とスライダーで可視化するWebアプリです。

## Development

```sh
pnpm install
pnpm dev
```

## Scripts

- `pnpm dev`: Vite development server
- `pnpm build`: TypeScript check and production build
- `pnpm lint`: ESLint
- `pnpm format`: Prettier write
- `pnpm format:check`: Prettier check
